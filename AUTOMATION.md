## 任务详情

### 证券早报推送 (每天 5:00)

**执行流程**:

1. **抓取新闻**: 从证券时报电子版抓取当日新闻
2. **AI摘要**: 使用 DeepSeek API 生成智能摘要
3. **飞书推送**: 自动将报告推送到飞书群
4. **本地保存**: 同时保存到 `history/` 目录

**报告内容**:

- **【头条要闻】**: 3-5条最重要的新闻
- **【值得关注】**: 5-8条相关新闻
- **【今日操盘提示】**: 1-2个交易要点
- **重点标注**: 🔥 自动标注 AI算力、半导体、军工等重点方向

### 美股隔夜简报推送 (每天 22:11)

**执行流程**:

1. **抓取新闻**: 从东方财富抓取美股导读和美股聚焦
2. **AI摘要**: 使用 DeepSeek API 生成智能摘要
3. **飞书推送**: 自动将报告推送到飞书群
4. **本地保存**: 同时保存到 `history/` 目录

**报告内容**:

- **三大指数与市场概况**: 美股指数走势分析
- **重要个股与板块**: 科技股、中概股表现
- **宏观与政策**: 地缘政治、美联储动态
- **对今日A股的影响**: 操盘参考和风险提示
