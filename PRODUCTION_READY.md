## 📱 飞书推送配置（可选）

如果你想在飞书群里接收买点信号：

### 获取飞书 Webhook

1. 打开飞书群
2. 右上角 → 群设置 → 机器人
3. 添加机器人 → 自定义机器人
4. 复制 Webhook 地址

### 配置到脚本

编辑 `cycle_stock_monitor.py`，找到第21行：

```python
FEISHU_WEBHOOK = "你的飞书Webhook地址"
```

替换为你的实际地址：

```python
FEISHU_WEBHOOK = "https://open.feishu.cn/open-apis/bot/v2/hook/..."
```

### 推送效果

启用后，每次运行时：
- 日报：三标的完整分析
- 警报：重要信号实时提醒
- 买点：关键时机立即通知
