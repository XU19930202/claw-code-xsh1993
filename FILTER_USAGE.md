# 公告下载过滤功能使用指南

## 功能说明

现在下载器支持**自动过滤**功能，可以精确控制下载的文件类型，自动排除摘要、募集说明书等不需要的文件。

## 过滤规则

### 1. `--filter all` （默认）
下载所有文件，排除摘要、募集说明书等无用文件。

```bash
python cninfo_downloader.py -c 603197 -k 年报 --download --filter all
```

**排除文件类型**: 摘要、募集说明书、转让说明书、更正

---

### 2. `--filter report` ⭐ 推荐
**只下载正式的财务报告**（年报、半年报、季报），排除所有摘要版本。

```bash
# 下载保隆科技所有报告
python cninfo_downloader.py -c 603197 -k 年报 --download --filter report -n 50

# 下载某公司2024-2025年的所有报告
python cninfo_downloader.py -c 600900 -k 报告 --download --filter report --start 2024-01-01 --end 2025-12-31
```

**包含文件类型**:
- ✓ 年度报告
- ✓ 半年度报告
- ✓ 第一季度报告
- ✓ 第三季度报告

**排除文件类型**:
- ✗ 年度报告摘要
- ✗ 半年度报告摘要
- ✗ 募集说明书
- ✗ 转让说明书

---

### 3. `--filter annual`
**只下载年度报告**（最全面的财务信息）

```bash
# 下载贵州茅台所有年度报告
python cninfo_downloader.py -c 600519 --download --filter annual -n 10
```

**包含**: 年度报告
**排除**: 半年报、季报、摘要

---

### 4. `--filter semi`
**只下载半年度报告**

```bash
# 下载中兴通讯所有半年报
python cninfo_downloader.py -c 000063 --download --filter semi -n 20
```

**包含**: 半年度报告
**排除**: 年报、季报、摘要

---

### 5. `--filter quarterly`
**只下载季度报告**（一季报和三季报）

```bash
# 下载某公司最近的季度报告
python cninfo_downloader.py -c 601966 --download --filter quarterly -n 10
```

**包含**:
- ✓ 第一季度报告
- ✓ 第三季度报告

**排除**: 年报、半年报、摘要

## 过滤规则详解

### 排除关键词（所有 filter 类型都适用）

以下关键词会被**自动排除**，即使匹配了过滤规则：

| 关键词 | 说明 | 案例 |
|--------|------|------|
| 摘要 | 简化版本，数据不完整 | 2024年年度报告摘要 |
| 募集说明书 | 融资相关文件 | 科创板首次公开发行募集说明书 |
| 转让说明书 | 股权转让文件 | 协议收购之被收购方转让说明书 |
| 更正 | 更正公告，非原始报告 | 2024年年度报告更正 |

## 高级技巧

### 1. 结合日期范围过滤
```bash
# 只下载 2024-2025 年的正式报告
python cninfo_downloader.py -c 600519 \
  --download \
  --filter report \
  --start 2024-01-01 \
  --end 2025-12-31 \
  -n 100
```

### 2. 结合关键词过滤
```bash
# 下载所有"年报"相关的正式文件
python cninfo_downloader.py -k "年报" \
  --download \
  --filter report \
  -n 100 \
  -o ./reports
```

### 3. 分类查询 + 过滤
```bash
# 从年报分类中只下载年度报告
python cninfo_downloader.py --cat 年报 \
  --download \
  --filter annual
```

## 常见问题

**Q: 为什么下载后的文件数量少于搜索结果？**
A: 因为过滤了摘要、募集说明书等不需要的文件。这是正常的！

**Q: 如何下载所有类型的文件，包括摘要？**
A: 使用 `--filter all`，但仍会排除明显不需要的文件如募集说明书等。

**Q: 能否自定义过滤规则？**
A: 目前提供了 5 种预设规则。如需自定义，可修改脚本中的 `should_download()` 函数。

**Q: 下载速度很慢怎么办？**
A: 这是正常的，因为涉及网络请求。程序会自动控制请求频率（0.3秒间隔）以避免被限流。
