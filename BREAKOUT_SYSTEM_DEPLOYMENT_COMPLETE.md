### 2️⃣ 决策树模块 (`breakout_decision_tree.py`)
✅ **功能**: 突破后买点判断
- 5日追踪期内自动分析
- 三条路径：放量涨→买、缩量企稳→买、放量跌→放弃
- 资金流交叉验证
- 逐日详细记录

✅ **买入信号判断**:
```
🟢 BUY (买入)
├─ 路径A：放量涨 (质量:标准)
├─ 路径A：缩量涨 (质量:优质)
└─ 路径B：回踩企稳 (质量:最优)

🟡 PULLBACK_WAIT (等待)
└─ 缩量回踩中，等待企稳

🔴 ABANDON (放弃)
├─ 放量下跌 (虚假突破)
└─ 连续跌破 MA20
```

✅ **运行方式**:
```bash
# 追踪昨天的突破信号
python breakout_decision_tree.py
```
