## 测试脚本使用指南

### 快速开始

#### 第1步：获取 Tushare Token

1. **访问 Tushare 官网**：https://tushare.pro
2. **注册账户**（如果还没有）
3. **获取 Token**：
   - 登录后，点击右上角"用户"→"个人主页"
   - 找到"Tushare Token"，复制你的 Token
   - Token 格式类似于：`abcdef123456...`

#### 第2步：配置脚本

编辑 `test_cycle_monitor.py`，找到第16行：

```python
TUSHARE_TOKEN = "替换为你的Tushare_Token"
```

替换为：

```python
TUSHARE_TOKEN = "你从Tushare获得的真实Token"
```

#### 第3步：确保依赖安装

```bash
pip install tushare numpy requests
```

#### 第4步：运行测试

```bash
cd c:/Users/Lenovo/WorkBuddy/20260311213700
python test_cycle_monitor.py
```

### 常见问题

**Q1：Token 怎么获取？**
- 访问 https://tushare.pro，注册免费账户
- 登录后在个人主页可以看到你的 Token

**Q2：脚本报错："获取数据失败"**
- 检查 Token 是否正确
- 检查网络连接
- 确保安装了 tushare：`pip install tushare --upgrade`

**Q3：需要付费吗？**
- 基础数据免费！获取日线、估值数据不需要付费
- Tushare 有免费额度，足够日常监控

**Q4：能否每天自动运行？**
- 可以！使用 WorkBuddy 自动化功能
- 或者用操作系统的定时任务（Windows 任务计划）
