## 🔧 命令参考

### 基础选项
| 选项 | 说明 | 示例 |
|------|------|------|
| `-c` | 股票代码 | `-c 600519` |
| `-k` | 关键词 | `-k 年报` |
| `--cat` | 公告分类 | `--cat 年报` |
| `-n` | 返回数量 | `-n 50` |
| `--download` | 下载公告 | `--download` |
| `-o` | 下载目录 | `-o ./mydir` |
| `--start` | 开始日期 | `--start 2024-01-01` |
| `--end` | 结束日期 | `--end 2024-12-31` |

### 常用组合
```bash
# 1. 按股票+关键词查询
python cninfo_downloader.py -c 000063 -k 年报

# 2. 按日期范围查询
python cninfo_downloader.py -k 重大资产重组 --start 2024-01-01 --end 2024-12-31

# 3. 查询并直接下载
python cninfo_downloader.py -c 600519 --cat 年报 --download -o ./annual

# 4. 按分类查询
python cninfo_downloader.py --cat 可转债 -k 赎回 -n 20
```

## 🚨 故障排查

### 问题1：搜索返回 0 条结果
- 原因: 参数格式错误或无匹配结果
- 解决: 尝试其他关键词或查看是否需要带交易所后缀

### 问题2：下载失败
- 原因: 网络不稳定或文件太大
- 解决: 脚本会自动跳过失败文件，可重新下载
