# 两个脚本快速对比与协同使用

## 🎭 两个脚本的角色

### 脚本1：cement_coal_tracker.py（周期追踪官）
**核心问题**：行业周期现在在哪里？
```
监控内容：
  ├─ 动力煤价格 → 成本端信号
  ├─ 水泥价格   → 需求端信号
  ├─ 海螺股价   → 市场预期
  └─ PE位置     → 利润周期
  
关键指标：煤价月跌幅、水泥价趋势、PE位置
输出周期：日报（对标日线行情）
```

**何时触发绿灯**：
- ✅ 煤价月跌幅 > 5% (成本下降)
- ✅ 水泥价格上升 (需求回暖)
- ✅ PE 在 10-15 之间 (中等位置)

**数据示例**：
```
📊 当前周期状态：
  煤价: 801.4 (月涨跌 +0%) → ❌ 没有下降信号
  水泥价: 自动获取失败    → ⚠️ 数据待确认
  海螺PE: 15.1 (中等)    → ✅ 周期中段
  
综合判断: ⚠️ 周期未完全确认
```

---

### 脚本2：conch_valuation_monitor.py（估值修复官）
**核心问题**：海螺的估值现在有没有修复空间？
```
监控内容：
  ├─ PB估值    → 价格端（底部回升）
  ├─ 技术面    → 确认端（MA20/MA60）
  └─ 基本面    → 支撑端（利润增长）
  
关键指标：PB分位数、MA突破、季报增速
输出周期：日报（对标收盘信号）
```

**何时触发绿灯**：
- ✅ PB < 0.8 (低估区间)
- ✅ MA20 拐头向上 (技术确认)
- ✅ 季报利润环比增长 (基本面支撑)

**数据示例**：
```
📊 当前估值状态：
  PB: 0.698 (1年范围0.601~0.772) → ✅ 低估区间
  PB趋势: ↘️ 短期回落              → ⚠️ 趋势未明
  MA20: 25.23 (价格在下方)         → ⚠️ 等待突破
  利润: 2025H1 +31%                → ✅ 拐点确认
  
综合评分: 3/6 → 🟡 修复进行中
```

## 🚦 三信号灯系统

### 信号灯1：周期灯（cement_coal_tracker）

| 颜色 | 含义 | 触发条件 |
|------|------|---------|
| 🟢 绿 | 周期向上 | 煤价↓ + 水泥价↑ + PE中等 |
| 🟡 黄 | 周期中段 | 部分指标满足 |
| 🔴 红 | 周期向下 | 煤价↑ + 水泥价↓ + PE低 |

### 信号灯2：估值灯（conch_valuation_monitor）

| 颜色 | 含义 | 触发条件 |
|------|------|---------|
| 🟢 绿 | 估值修复启动 | PB低 + MA20↑ + 利润↑ (得分≥5) |
| 🟡 黄 | 估值修复进行中 | 部分条件满足 (得分3-4) |
| 🔴 红 | 估值修复未启动 | 大部分条件未满足 (得分<3) |

### 组合信号（两个灯的组合）

| 周期灯 | 估值灯 | 综合信号 | 操作建议 |
|-------|-------|--------|--------|
| 🟢 | 🟢 | 🟢🟢 双绿 | **强烈买入** |
| 🟢 | 🟡 | 🟢🟡 一绿一黄 | **可以买入** |
| 🟡 | 🟢 | 🟡🟢 一黄一绿 | **谨慎买入** |
| 🟡 | 🟡 | 🟡🟡 双黄 | **继续观察** |
| 🟢 | 🔴 | 🟢🔴 背离 | **避免操作** |

## 📋 每日操作检查表

### 早间（9:30-10:00）
```
□ 运行 cement_coal_tracker.py
  ├─ 检查动力煤昨日收盘价
  ├─ 查看水泥价格（需手动验证）
  └─ 记录海螺股价和PE

□ 记录结果
  └─ 保存到 morning_check.txt
```

### 收盘后（16:30-17:00）
```
□ 再次运行 cement_coal_tracker.py
  └─ 对比早间和收盘数据

□ 运行 conch_valuation_monitor.py
  ├─ 检查PB位置变化
  ├─ 查看MA信号
  └─ 验证基本面数据

□ 综合判断
  ├─ 周期灯状态
  ├─ 估值灯状态
  └─ 组合信号判断
```

## 🛠️ 快速命令

```bash
# 同时查看两个脚本结果
python cement_coal_tracker.py > /tmp/1.txt && python conch_valuation_monitor.py > /tmp/2.txt && cat /tmp/1.txt && echo "---" && cat /tmp/2.txt

# 保存对比分析
python cement_coal_tracker.py --output cycle.txt
python conch_valuation_monitor.py --output valuation.txt

# 查看是否有重大信号
python conch_valuation_monitor.py | grep "🔥\|🚨"
```

## 📌 核心记住

| 脚本 | 简称 | 问题 | 答案是什么时买 |
|------|------|------|----------------|
| cement_coal_tracker | 周期官 | 行业在哪 | 煤↓水↑PE中 |
| conch_valuation_monitor | 估值官 | 估值在哪 | PB低MA↑利润↑ |
| 组合判断 | 决策官 | 现在买不买 | 两个都绿才买 |

---

**记住**：两个脚本一个管"大盘"（周期），一个管"个股"（估值）。  
只有大盘周期向上 + 个股估值有空间，才是真正的买点！

✅ **双绿灯 = 黄金买点**  
🟡 **一绿一黄 = 观察期**  
🔴 **任何红灯 = 避免操作**
