### 🎯 第2层：追踪 (Tracking Phase)

```
【昨天突破 → 今天追踪 5 日 → 判断买入时机】

输入: 昨天的突破信号 (来自扫描历史)
  ↓
5日追踪期内，逐日分析走势:

【路径A】放量涨
  ├─ 放量上涨 → ✅ 买入 (质量:标准)
  └─ 缩量上涨 → ✅ 买入 (质量:优质)

【路径B】缩量回踩
  ├─ 缩量跌但未破MA → 继续观察
  ├─ 出现十字星/小阳 → 等待企稳
  └─ 企稳+收上MA → ✅ 买入 (质量:最优)

【路径C】放量跌
  ├─ 放量下跌 → ❌ 放弃 (虚假突破)
  └─ 连续跌破MA → ❌ 放弃 (支撑无效)

  ↓
输出: BUY / PULLBACK_WAIT / ABANDON
  ↓
资金流验证: 主力资金净流入确认
```

**关键判定**:
- `VOLUME_RATIO_THRESHOLD = 1.2` (放量判定)
- `SHRINK_RATIO_THRESHOLD = 0.8` (缩量判定)
- `DOJI_BODY_RATIO = 0.003` (十字星)
- `SMALL_YANG_MAX = 0.02` (小阳线)
- `TRACK_DAYS = 5` (追踪期)
- `ENABLE_MONEYFLOW = True` (资金流验证)

**输出示例**:

## 操作纪律 (Trading Discipline)

```
【买入】
  1. 收到 🟢 BUY 信号
  2. 买入 1/3 头寸
  3. 设定止损 = MA20 - 1%

【加仓】
  1. 站稳 MA20 连续 3-5 天
  2. 加仓 1/3
  3. 调整止损到成本价

【再加】
  1. 创新高 (高于突破价 3% 以上)
  2. 加仓最后 1/3
  3. 跟踪止损

【止损】
  1. 3 日内跌回 MA20 下方
  2. 或 2 日内跌幅 > 5%
  3. 立即止损全部
```
