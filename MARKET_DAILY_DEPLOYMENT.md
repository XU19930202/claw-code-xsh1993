# A股市场简报部署说明

## 概述
已成功部署A股市场简报自动化工具,支持每日早晚两次自动推送。

## 部署内容

### 1. 核心脚本
**文件**: `market_daily.py`

**功能**:
- 指数行情: 上证指数、深证成指的价格和涨跌幅
- 全A市场: 成交额、上涨家数、下跌家数
- 重点个股: 5只监控股票(翱捷科技、*ST星光、*ST天龙、中旗新材、宝莱特)的价格、涨跌幅、成交额比
- ETF信息: 华泰柏瑞沪深300ETF价格及份额变动

**数据源**: Tushare API

**推送方式**: 飞书机器人Webhook

### 2. 自动化任务配置

#### 任务1: A股早间简报推送
- **ID**: `a`
- **执行时间**: 每天早上 7:00
- **RRULE**: `FREQ=DAILY;BYHOUR=7;BYMINUTE=0`
- **状态**: ✅ ACTIVE
- **执行记录**: `.codebuddy/automations/a/memory.md`

#### 任务2: A股收盘简报推送
- **ID**: `a-2`
- **执行时间**: 每天下午 16:00
- **RRULE**: `FREQ=DAILY;BYHOUR=16;BYMINUTE=0`
- **状态**: ✅ ACTIVE
- **执行记录**: `.codebuddy/automations/a-2/memory.md`

### 3. 智能交易日判断
脚本内置了交易日判断逻辑:
- 只有当日是交易日才会推送
- 非交易日(周末、节假日)会自动跳过推送
- 历史记录保存在 `history/market_YYYYMMDD.md`

## 配置说明

所有配置文件使用统一的 `config.yaml`,包括:
- 飞书Webhook地址
- LLM API配置(如有需要)

如需修改监控个股列表,编辑 `market_daily.py` 中的 `WATCH_STOCKS` 变量。

## 注意事项

1. 确保Tushare API Token有效(已配置在脚本中)
2. 确保config.yaml中已配置飞书Webhook
3. 脚本会自动跳过非交易日推送
4. ST股票代码可能因更名而变化,需要定期检查更新
