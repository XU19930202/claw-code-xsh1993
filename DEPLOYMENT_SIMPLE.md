由于 CLI 工具的复杂性,我们推荐使用**控制台手动上传**的方式,这是最简单直接的方法。

---

### 步骤2: 打开腾讯云控制台

访问: https://console.cloud.tencent.com/scf

---

### 步骤3: 创建云函数

1. **点击"新建函数"**

2. **填写函数信息**
   - 函数名称: `hotspot_tracker`
   - 运行环境: `Python 3.9`
   - 地域: 选择你需要的地域(如:广州)
   - 函数代码: 选择"本地上传压缩包"

3. **上传代码**
   - 点击"上传压缩包"
   - 选择 `hotspot_tracker_scf.tar.gz`
   - 等待上传完成

4. **配置基本设置**
   - 执行方法: `index.main_handler`
   - 内存: `256 MB`
   - 超时时间: `300 秒`

5. **点击"完成"**

---

### 步骤4: 配置环境变量

1. 在函数详情页,点击左侧"配置"→"环境变量"

2. 点击"编辑",添加以下环境变量:

   ```
   FEISHU_WEBHOOK=https://open.feishu.cn/open-apis/bot/v2/hook/ee48166c-c506-46f0-b73a-36fcbbcd0ac6
   LLM_API_KEY=sk-f6e0ad44a3c94143aae333a1172fee48
   ```

3. 点击"保存"

---

### 步骤5: 配置定时触发器

1. 在函数详情页,点击左侧"触发管理"

2. 点击"创建触发器"

3. 填写触发器信息:
   - 触发方式: 定时触发器
   - Cron表达式: `0 0 8 * * * *`
   - 说明: 每天早上8点执行

4. 点击"提交"

**Cron表达式说明**: `秒 分 时 日 月 星期 *`
- `0 0 8 * * * *` = 每天 8:00:00 执行

---

### 步骤6: 测试函数

1. 在函数详情页,点击"测试"按钮

2. 点击"运行测试"

3. 查看执行日志:
   - 点击左侧"日志"→"日志查询"
   - 查看最新的日志输出

4. 检查飞书群是否收到推送

---

### 步骤7: 修改执行时间(可选)

如果需要修改执行时间:

1. 点击左侧"触发管理"

2. 找到定时触发器,点击"编辑"

3. 修改 Cron表达式:

   常用示例:
   - `0 0 9 * * * *` - 每天 9:00
   - `0 30 8 * * * *` - 每天 8:30
   - `0 0 */6 * * * *` - 每6小时执行一次
   - `0 0 12 * * 1-5` - 工作日 12:00

4. 点击"保存"

## 📝 其他部署方式

如果你熟悉命令行工具,也可以考虑以下方式:

### 方式1: 使用腾讯云 CLI (scfcli)

```powershell
# 安装
pip install scfcli

# 配置
scf configure

# 部署
scf deploy
```

**注意**: CLI 方式需要配置多个文件,相对复杂。

### 方式2: 使用 Serverless Framework

```powershell
# 安装
npm install -g serverless

# 配置
serverless config credentials --provider tencentcloud

# 部署
serverless deploy
```

## 🆘 常见问题

### Q1: 上传失败?

**A**: 检查压缩包是否完整,重新上传。

### Q2: 函数执行失败?

**A**:
1. 查看日志了解错误信息
2. 检查环境变量是否配置正确
3. 确认网络连接正常

### Q3: 没有收到飞书推送?

**A**:
1. 检查飞书 Webhook 地址是否正确
2. 查看函数执行日志
3. 确认飞书机器人未被禁用
