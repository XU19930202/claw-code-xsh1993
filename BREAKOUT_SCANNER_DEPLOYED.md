## 📋 评分说明

每只股票评分 0-7 分：

| 条件 | 分值 |
|------|------|
| 强放量(≥2.0x) | 3 分 |
| 中等放量(1.5-2.0x) | 2 分 |
| 涨幅≥3% | 2 分 |
| 涨幅1-3% | 1 分 |
| 均线向上 | 1 分 |
| 站在更长均线上 | 1 分 |

**高分优先**：6-7 分是强信号，值得关注

## 💡 操作纪律

根据扫描结果采取行动：

```
✅ 放量突破当天 → 买入 1/3 仓位
✅ 站稳 3-5 天不破 → 加仓 1/3 仓位
✅ 3 天内跌回均线下 → 止损全部
❌ 未放量 → 只观察不操作
```

## 🔧 技术细节

### 扫描规则

```
规则1：首次站上均线
  - 昨天收盘 ≤ 昨日均线
  - 今天收盘 > 今日均线

规则2：放量确认
  - 今日成交量 ≥ 5 日均量 × 1.5
```

### 排除条件

脚本自动排除以避免噪声：
- ❌ ST 股（停牌风险）
- ❌ 退市股
- ❌ 北交所股票
- ❌ 上市不满 6 个月的次新股
