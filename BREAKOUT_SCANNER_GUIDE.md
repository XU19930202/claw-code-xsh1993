# 🔍 全市场均线突破扫描器 - 部署指南

## 📌 功能概览

这是一个**全市场扫描器**，每天收盘后自动扫描全部 A 股（约 4000+ 只），找出那些：
- ✅ **首次站上某条均线**（昨天在下方，今天在上方）
- ✅ **伴随放量**（成交量 ≥ 5 日均量 × 1.5 倍）

然后推送到飞书，帮助你快速发现市场中的买点机会。

---

## 🚀 快速开始

### 命令格式

```bash
# 扫描 MA20 突破（默认）
python breakout_scanner.py

# 扫描 MA60 突破
python breakout_scanner.py --ma 60

# 扫描 MA120 突破
python breakout_scanner.py --ma 120

# 只打印不推送飞书（测试用）
python breakout_scanner.py --no-push

# 组合使用
python breakout_scanner.py --ma 60 --no-push
```

## 📋 评分系统（满分 7 分）

每只发现的股票都会获得一个 0-7 分的评分：

| 项目 | 分值 | 说明 |
|------|------|------|
| **放量** | 0-3 分 | 2.0x 以上强放量(3分) / 1.5-2.0x 放量(2分) / 低于1.5x(0分) |
| **涨幅** | 0-2 分 | ≥3%涨幅(2分) / 1-3%涨幅(1分) / <1%(0分) |
| **均线方向** | 0-1 分 | 均线向上(1分) / 均线向下(0分) |
| **长期均线** | 0-1 分 | 同时站在更长均线上(1分) / 未站上(0分) |

**评分高的优先关注**（6 分以上是强信号）

## 💡 操作纪律

根据脚本的建议，实际操作时遵循：

```
放量突破当天 → 买入 1/3 仓位
站稳 3-5 天不破均线 → 加仓 1/3 仓位
3 天内跌回均线下 → 止损全部
未放量的信号 → 只观察不操作
```

## ⚠️ 注意事项

1. **数据延迟**
   - 扫描结果基于当日收盘后的数据
   - 不支持实时中文扫描

2. **误报风险**
   - 未放量的信号要谨慎（只观察不操作）
   - 单纯均线突破不一定能赚钱
   - 还需结合**基本面**和**风险管理**

3. **网络/API 限制**
   - 首次扫描可能需要 35-40 分钟
   - 如中途中断，需要重新运行
   - Token 充足的话不用担心费用

4. **系统资源**
   - 扫描时会占用网络和 CPU
   - 建议安排在交易日收盘后（17:30 后）
   - 不要在盘中运行（数据不完整）

## 📞 常见问题

### Q1：为什么扫描这么慢？

**原因**：需要获取 4000+ 只股票的 150 天日线数据，计算均线和指标

**解决**：
- 这是正常的，第一次会慢一点
- 后续系统会缓存数据，稍微快一点
- 可以并行运行多个进程（高级用法）

### Q2：能否只扫描某些行业？

**目前不支持**，但可以修改脚本中的 `get_all_stocks()` 函数来指定行业。

### Q3：推送到飞书失败怎么办？

```bash
# 测试 Webhook 连接
python test_feishu_webhook.py

# 或先运行不推送版本
python breakout_scanner.py --no-push
```

### Q4：能否改变扫描时间？

可以在 WorkBuddy 自动化中修改 RRULE：
```
改为 18:00: FREQ=DAILY;BYHOUR=18;BYMINUTE=0
改为 19:30: FREQ=DAILY;BYHOUR=19;BYMINUTE=30
```
