# 巨潮资讯网公告下载器 - 使用指南

## 📋 功能介绍

这是一个功能完整的巨潮资讯网公告爬虫和下载器，支持：
- ✅ 按股票代码搜索公告
- ✅ 按关键词搜索（重大资产重组、年报、融资、解禁等）
- ✅ 按公告分类过滤（35+ 种分类）
- ✅ 按日期范围筛选
- ✅ 批量下载 PDF
- ✅ 交互式和命令行两种模式

---

## 🚀 快速开始

### 模式一：交互模式（推荐）

```bash
cd c:/Users/Lenovo/WorkBuddy/20260311213700
python cninfo_downloader.py
```

然后按提示输入命令（s 搜索、d 下载、n 翻页等）

---

### 模式二：命令行快速查询

#### 搜索指定股票的所有公告

```bash
python cninfo_downloader.py -c 000063
```

#### 搜索特定关键词公告

```bash
python cninfo_downloader.py -k 重大资产重组 -n 20
```

#### 按代码+关键词搜索

```bash
python cninfo_downloader.py -c 000063 -k 年报
```

#### 按分类搜索并下载

```bash
python cninfo_downloader.py -c 600519 --cat 年报 --download
```

#### 指定日期范围搜索

```bash
python cninfo_downloader.py -c 000063 --start 2024-01-01 --end 2024-12-31 --download
```

## ⚙️ 高级用法

### 交互模式下的高级搜索

```
> s 000063                              # 按代码搜索
> s 000063 年报                         # 代码 + 关键词
```

```
> d 1              # 下载序号1
> d 1 3 5          # 下载序号1、3、5
> d 1-10           # 下载序号1到10
> da               # 下载当前页所有
```

## 🔧 故障排查

### 问题1：网络连接失败

**原因**: 巨潮网站可能临时不可用或网络超时  
**解决**: 重试或稍后再试，脚本有 15 秒超时限制

### 问题2：文件名包含非法字符

**原因**: 某些公告标题包含特殊字符  
**解决**: 脚本已自动清理，会将 `\ / : * ? " < > |` 替换为 `_`

### 问题3：下载失败或文件不完整

**原因**: 网络中断或对方服务器问题  
**解决**: 文件会被自动删除，可重新下载
