| VIX恐慌指数监控(早间) | 早上 5:00 | vix | 早间监控 |
| VIX恐慌指数监控(午前) | 上午 10:00 | vix-2 | 午前监控 |
| VIX恐慌指数监控(午后) | 下午 14:00 | vix-3 | 午后监控 |
| VIX恐慌指数监控(晚间) | 晚上 18:00 | vix-4 | 晚间监控 |

### 功能说明
1. **多数据源**: Alpha Vantage / Yahoo Finance / Investing.com
2. **智能预警**: 仅当 VIX > 20 时推送预警
3. **分级提示**: 根据VIX水平提供不同预警级别
   - VIX < 15: 极低 😌 (市场情绪平静)
   - VIX < 20: 较低 🙂 (市场情绪稳定)
   - VIX < 25: 中等 😐 (波动性上升)
   - VIX < 30: 较高 😟 (恐慌情绪增加)
   - VIX < 40: 很高 😰 (恐慌情绪显著)
   - VIX >= 40: 极高 😱 (市场极度恐慌)

4. **详细数据**: 包含开盘价、最高价、最低价、前值对比等

### 数据输出
- **历史记录**: `history/vix_YYYY-MM-DD.log`
- **日志文件**: `vix_monitor.log`
