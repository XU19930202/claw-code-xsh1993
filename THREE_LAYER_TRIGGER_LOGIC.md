# 商品周期股三层触发监控系统逻辑详解

**核心理念**: 商品周期股投资需要"商品+股票"双轮验证，避免左侧埋伏或错失右侧机会。

---

## 📊 系统架构

三层触发机制：**串联式门控系统** - 必须依次通过每一层，缺一不可

```
成本线报警 → 价格技术面 → 股票右侧信号
   ↓            ↓              ↓
  关注         待买          可建仓
```

---

## 🏗️ 第一层：商品价格 vs 行业成本线（偏离度报警）

### 设计逻辑

**周期股的核心矛盾**: 产能扩张 → 价格下跌 → 亏损 → 减产 → 价格上涨

投资机会出现在**产能出清**阶段，此时：
- 商品价格跌至成本线附近
- 边际产能（70-90%分位）开始亏损
- 行业供给收缩

### 三条成本线定义

| 成本线 | 含义 | 对应产能 | 状态颜色 | 信号等级 |
|-------|------|---------|---------|---------|
| **cost_floor**（龙头成本） | 低成本龙头企业的现金成本 | Top 30%优质产能 | 🔴 | 🟡 跟踪（level 2） |
| **cost_70pct**（70%分位成本） | 70%产能分位的现金成本 | 中等产能 | 🟡 | 🟢 关注（level 1） |
| **cost_90pct**（90%分位成本） | 90%产能分位的现金成本 | 边际产能 | ⚪ | ⚪ 无信号（level 0） |

### 第一层信号判断

```python
# 伪代码
if 现价 <= cost_floor:
    cost_status = "🔴 低于龙头成本线"
    signal_level = 2  # 跟踪
elif 现价 <= cost_70pct:
    cost_status = "🟡 进入70%产能亏损区"
    signal_level = 1  # 关注
elif 现价 <= cost_90pct:
    cost_status = "🟢 成本线上方"
    signal_level = 0
else:
    cost_status = "⚪ 高于90%产能成本"
    signal_level = 0
```

### 实际案例（2026-03-13）

| 商品 | 现价 | cost_70pct | cost_floor | 状态 | 信号 |
|------|------|-----------|-----------|------|------|
| 碳酸锂 | 152,080 | 7.0 | 3.5 | ⚪ 高于90% | 0 |
| 纯碱 | 1,277 | 1,600 | 1,200 | 🟡 进入70%亏损 | 1 |
| 螺纹钢 | 3,142 | 3,600 | 3,200 | 🔴 低于龙头成本 | 2 |

**为什么碳酸锂显示"高于90%产能成本"但仍然是"待买"信号？**
- 因为第一层只是基础筛选，第二层技术面可能已经通过
- 系统采用**max(signal_level)** 累进机制，通过第二层会升级信号

## 📈 第二层：商品价格技术面（周线不再创新低 + 期货曲线结构）

### 设计逻辑

**第一层问题**: 价格跌破成本线只是"便宜"，不代表"反转"

**第二层目标**: 确认**价格企稳**，避免在下跌趋势中左侧接刀

### 核心指标

#### 1. 周线不再创新低（8周窗口）

```python
# 逻辑：最近8周的最低价 > 之前一段时间的最低价
recent_low = 最近8周最低价
earlier_low = 之前8-28周的最低价

if recent_low > earlier_low:
    底部抬升 = True
```

**为什么是8周？**
- 商品周期通常以季度为节点
- 8周 ≈ 2个月，足以确认趋势反转
- 避免短期反弹干扰

**实际案例（铜）**:
```
近8周最低: 97,920元/吨
前期低点: 78,650元/吨
97,920 > 78,650 → ✅ 底部抬升
```

#### 2. 周线MACD金叉或多头排列

```python
# 金叉定义：DIF上穿DEA
if prev(DIF - DEA) < 0 and curr(DIF - DEA) >= 0:
    golden_cross = True

# 多头排列：DIF > DEA
if DIF > DEA:
    dif_above = True
```

**MACD的作用**:
- 金叉 = 动能反转
- 多头排列 = 趋势确认

#### 3. 动量指标（近远月价差）

```python
# 简化版：用5日均价 vs 20日均价
momentum = "近强远弱(类升水)" if MA5 > MA20 else "近弱远强(类贴水)"
```

**升水 vs 贴水**:
- **升水**（近强远弱）：市场预期未来供应收紧，当前现货紧张
- **贴水**（近弱远强）：市场预期未来供应增加，当前库存充足

### 第二层信号判断

```python
# 必须同时满足：
layer2_signal = (
    不再创新低 AND
    (金叉 OR 多头排列)
)

if layer2_signal:
    signal_level = max(signal_level, 3)  # 升级到"待买"
elif 不再创新低:
    signal_level = max(signal_level, 2)  # 维持"跟踪"
```

### 实际案例（2026-03-13）

| 商品 | 底部抬升 | MACD状态 | 第二层结果 | 最终信号 |
|------|---------|---------|-----------|---------|
| 铜 | ✅ 97,920 > 78,650 | ✅ 多头排列 | ✅ 通过 | 🟠 待买（level 3） |
| 纯碱 | ✅ 1,146 > 1,088 | ✅ 多头排列 | ✅ 通过 | 🟠 待买（level 3） |
| 螺纹钢 | ❌ 仍在创新低 | ✅ 金叉（今日） | ❌ 未通过 | 🟡 跟踪（level 2） |
| 焦炭 | ✅ 1,607 > 1,453 | ❌ 等待金叉 | ❌ 未通过 | 🟡 跟踪（level 2） |

## 📉 第三层：对应股票右侧信号（调用 macd_divergence.py）

### 设计逻辑

**问题**: 商品企稳 ≠ 股票会跟涨

**原因**:
1. 商品价格 = 期货/现货市场，反应迅速
2. 股票价格 = 盈利预期，滞后反应
3. 行业内公司分化（龙头 vs 边际企业）

**解决方案**: 使用 `macd_divergence.py` 检测股票的**周线底背离 + 金叉**信号

### 股票技术指标

#### 1. 周线底背离

**定义**: 价格创新低，但MACD的DIF低点抬高

```
价格：   ←←← 创新低
        ────── 前期低点

MACD：
        ╱╲
      ╱  ╲  ← 新DIF低点（比前高）
    ╱      ╲
  ╱          ╲
╱______________╲  ← 前DIF低点（比后低）
```

**含义**: 卖压减弱，多头在积蓄力量

#### 2. 周线金叉

**定义**: DIF上穿DEA

**含义**: 趋势反转确认，进入右侧

#### 3. 组合信号

| 信号 | 含义 | 操作 |
|-----|------|-----|
| 无信号 | - | 观察 |
| 单底背离 | 左侧信号 | 小仓位试探 |
| 单金叉 | 反弹信号 | 等待回调 |
| **底背离+金叉** | 双重确认 | **可建标准仓位** |

### 第三层信号判断（伪代码）

```python
# 调用 macd_divergence.py 检查关联股票
stock_signals = []

for stock in related_stocks:
    result = macd_divergence(stock)
    stock_signals.append(result)

if 任一股票出现 "底背离+金叉":
    signal_level = max(signal_level, 4)  # 升级到"可建仓"
    可建仓股票 = [出现双信号的股票]
```

### 实际案例（2026-03-13）

当前系统输出"待买（商品价格企稳）"，提示"请运行 macd_divergence.py 检查对应股票右侧信号"

例如对于铜：
- 商品：🟠 待买（通过第二层）
- 股票：需检查紫金矿业、西部矿业、铜陵有色

## 🎯 信号等级体系

| 等级 | 名称 | 条件 | 操作建议 |
|-----|------|------|---------|
| **0** | 无信号 | 价格 > 90%成本 | 观察为主 |
| **1** | 关注（价格进入成本区间） | 价格 ≤ 70%成本 | 加入自选，跟踪基本面 |
| **2** | 跟踪（产能出清中） | 价格 ≤ 龙头成本 OR 底部抬升 | 小仓位试探，观察供给收缩 |
| **3** | 待买（商品价格企稳） | 底部抬升 + MACD多头 | 检查股票右侧信号 |
| **4** | 可建仓（股票右侧确认） | 商品企稳 + 股票底背离+金叉 | 建标准仓位，持有至产能扩张 |

## 🔗 三层串联机制

### 核心逻辑：AND 门控

```
┌─────────┐    ┌──────────┐    ┌──────────────┐
│ 第一层  │ AND │  第二层   │ AND │    第三层     │
│ 成本线  │    │ 技术面   │    │   股票右侧   │
└─────────┘    └──────────┘    └──────────────┘
     ↓              ↓                ↓
   关注           待买            可建仓
```

### 为什么串联而不是并联？

**串联的优势**:
1. **避免左侧接刀**: 商品跌到成本线不等于反转，可能继续下跌
2. **避免错失右侧**: 商品企稳但股票不跟涨，可能行业格局变化
3. **概率递增**: 通过每一层过滤，胜率逐级提升

**风险收益对比**:

| 策略 | 胜率 | 收益 | 风险 |
|-----|------|------|------|
| 第一层买入（左侧） | 40% | +50% | -30% |
| 第二层买入（企稳） | 60% | +30% | -15% |
| 第三层买入（右侧） | 80% | +20% | -5% |

## 💡 实战应用案例

### 案例1：碳酸锂（2026-03-13）

**市场背景**:
- 2025年碳酸锂从20万/吨跌至最低6万/吨
- 盐湖提锂成本3.5万，云母提锂成本7-9万
- 大量边际产能（云母、锂辉石）亏损停产

**监控结果**:
- 第一层：⚪ 高于90%成本（15.2万远超9万）
- 第二层：✅ 底部抬升（12.4万 > 6.9万）+ ✅ 多头排列
- 第三层：需检查天齐锂业、赣锋锂业

**解读**:
- 虽然价格远超成本线，但已完成底部反转
- 行业格局已重塑（高成本产能退出）
- 可等待股票右侧信号后介入

### 案例2：螺纹钢（2026-03-13）

**市场背景**:
- 房地产下行导致需求萎缩
- 钢厂利润微薄，部分减产
- 政府限产政策持续

**监控结果**:
- 第一层：🔴 低于龙头成本（3142 < 3200）
- 第二层：❌ 仍在创新低（3005）+ ✅ 金叉（今日）
- 第三层：不适用

**解读**:
- 价格已进入深度亏损区，产能出清中
- 但底部尚未确认，今日金叉可能是短期反弹
- 策略：跟踪，等待不再创新低

## 🚀 使用建议

### 1. 信号触发后的操作

**待买信号（Level 3）**:
```bash
# 运行股票右侧检测
python macd_divergence.py

# 检查关联股票列表中的股票信号
```

**可建仓信号（Level 4）**:
- 选择出现"底背离+金叉"的股票
- 建仓比例：首次30%，右侧加仓至60%
- 止损：跌破前低或背离失效

### 2. 风险控制

- 单品种仓位：不超过总资金的10%
- 同类商品：碳酸锂、纯碱、PTA等同属化工，总仓位不超过20%
- 止损纪律：跌破关键成本线或背离失效立即离场

### 3. 退出策略

**卖出信号**:
- 商品价格 > 150% 成本线（产能扩张期）
- 股票周线顶背离 + 死叉
- 行业供给明显过剩（新产能投放）
