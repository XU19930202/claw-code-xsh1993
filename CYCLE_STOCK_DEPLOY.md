2. **海螺水泥** (600585.SH) - 水泥龙头
   - 等待信号：MA20放量突破

   - 等待信号：缩量企稳 + 重回MA20

## 📊 信号解释

### 信号级别
- **级别 0**：无信号（继续等待）
- **级别 1**：关注信号 🟡
  - 接近MA20突破
  - 均线走好但未确认
  - 缩量企稳迹象
  
- **级别 2**：重要信号 🟠
  - 站上MA20（短期趋势转好）
  - 站上MA60（中期趋势转好）
  
- **级别 3**：立即行动 🔴
  - **放量突破MA20**（最关键买点）
  - 这是最重要的入场信号

### 技术指标说明

| 指标 | 含义 | 买点含义 |
|------|------|--------|
| **MA20** | 20日均线 | 短期支撑/阻力 |
| **MA60** | 60日均线 | 中期支撑/阻力 |
| **MA120** | 120日均线 | 长期支撑/阻力 |
| **量比** | 今日成交量/5日均量 | >1.5为放量 |
| **MA20方向** | MA20的30日趋势 | 向上=技术面向好 |
| **PE/PB** | 估值指标 | 低位更安全 |

## 🔧 常见问题

### Q1：Tushare Token 获取失败？
A：
1. 确保网络连接正常
2. Token可能已过期，重新从 tushare.pro 获取
3. 检查余额是否充足（免费版可能有限制）

### Q2：飞书推送失败？
A：
1. 检查 Webhook URL 是否正确
2. 确认机器人未被禁用
3. 检查网络连接
4. 查看飞书群组权限设置

### Q3：数据获取超时？
A：
1. 检查网络连接
2. Tushare 服务可能繁忙，稍后重试
3. 增加 timeout 参数

### Q4：如何修改监控标的？
A：
修改 `STOCKS` 列表：
```python
STOCKS = [
    {
        "code": "新代码.SH",  # 上交所用.SH，深交所用.SZ
        "name": "股票名称",
        ...
    }
]
```
